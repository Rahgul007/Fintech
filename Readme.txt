1.install python 3.7+ version in your system.

2.check your system python installed properly 
	
	e.g->python --version

3.install virtual enviornment in your project folder.

	eg->py -m venv myvenv->(your env name whatever it is)

4.activate your virtual env 
	
	e.g->myvenv\Scripts\activate

5.install requirements.txt file

	e.g->pip install -r requirements.txt->it will install all dependency python files

6.Run python code.
	
	e.g->python app.py

7.import json file in postman for getting api urls

8.import .db files in your database(mysql)
